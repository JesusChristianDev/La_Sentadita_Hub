# Bloque B04 — Documentos (MVP) + Reconfirmación password cada 10 min

## Objetivo
- Librería de documentos por restaurante.
- Acciones sensibles (ver/descargar/subir/borrar) requieren reconfirmación de contraseña cada 10 min.

## Alcance MVP
- Tabla documents + storage bucket privado.
- Listado / subida / descarga (signed URL).
- RLS por rol:
  - admin/office: todos los restaurantes
  - manager: su restaurante
  - employee: su restaurante (lectura)

## Seguridad
- Cookie httpOnly reauth_at con timestamp.
- Gate server-side: si reauth_at > 10 min -> redirige a /reauth?next=...
